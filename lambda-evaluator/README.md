# lambda evaluator

Features 
1) evaluation according normal strategy        -> function normalEvaluation <br /> 
2) evaluation according applicative strategy   -> function applicativeEvaluation <br /> 
3) eta conversion                              -> function optimilization <br /> 
4) expand macros in lambda expression          -> function expand <br /> 
5) lambda function that takes n parametrs <br /> 

## Usage

1) Function normalEvaluation takes two parametr. First parametr is lambda expression without macros, second is flag. 0 -> don't print evaluation of lambda , 1-> print evaluation <br /> 
2) Function applicativeEvaluation takes one parametr. First parametr is lambda expression without macros <br /> 
3) Function optimilization takes one parametr. First parametr is lambda expression without macros <br /> 
4) Function expansion takes one parametr. First parametr is lambda expression with macros <br />
